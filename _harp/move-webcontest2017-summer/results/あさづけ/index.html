N高 動くWebページコンテスト 2017夏

このページのテーマは
「 N予備校で、今の 高校生が学ぶウェブ技術に 追いつこう！」です。

  • 自身のプログラム経験
    • HTML　　　:　文字と画像だけの簡単なホームページが作れた程度。
    CSS. 　　　:　コピペするとなぜか使えるという噂を聞いていた程度。
    Javascript　:　無し。
  • 予備校での学習時間・・
    • 　　　　ほぼ一ヶ月！
    各コンテンツ説明
  • カレンダー：new Date()関数から今月の「一日(ついたち)」の曜日データを取得して、
    　　　　　　その曜日を基準に、各曜日を設定したテーブルの曜日枠へ出力。
    　　　　　　各月で曜日がスライドして曜日と日にちを合わせています。
    　　　　　　３０日の月は３１日を数字から「－」に変更。
    　　　　　　２月は２９～３１日枠の数字を「－」に変更。

  • 時計の秒針：new Date()関数から現在時間の秒を取得して、
    　　　　　　秒に６をかけて角度数値に変換してからrotateZ関数に代入してループ。

  • 動く画像　：javascriptでcssの画像配置位置数値を増減させています。
    　　　　　　左右～px以上、以下の時 true,falseとif文で指示して方向をかえています。

  • ぱんださん：各コメントを配列にいれて、一つしゃべるごとに引数を＋１、
    　　　　　　配列のlengthいっぱいまできたら最初にもどるの関数を、
    　　　　　　インターバル関数で回しています。

  • おみくじ　：０～４のランダム数を生成して、各数値に大凶～大吉をあてがい
    　　　　　　onclick関数で出力。（いいところ診断の使いまわし）

  • FizzBuzz：　予備校テキストで作ったフィズバズと、そのあとの授業で習ったもので
    　　　　　　こねくりまわしたものです。
    　　　　　　ランダム値関数で出たランダム値をfizzbuzz関数で判断させ、
    　　　　　　その判断させて出た値とその時に押したボタンと比較して
    　　　　　　if文で正解不正解を出しています。


  • ヘッダー部分、ツィッターチェックとフィズバズの表示
    　javascriptでCSSのdisplay値：noneとblockを交互に入れ替え。
    　ページを開いて最初だけ、ボタンをダブルクリックしないと表示されない不具合は、
    　今の私の技量では直せませんでした(;´∀｀)がーん
